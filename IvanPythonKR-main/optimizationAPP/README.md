# Optimization App

Графічний застосунок для чисельної мінімізації багатовимірних **унімодальних** функцій різними методами.  
Програма дозволяє обирати цільову функцію, метод оптимізації, задавати початкову точку та параметри зупинки, а також спостерігати **табличний** і **графічний** процес збіжності.

---

## Можливості

- Вибір **будь-якого з реалізованих методів**:
  1. Метод Коші (градієнтний спуск з лінійним пошуком)
  2. Метод Флетчера–Рівза (градієнтний метод спряжених напрямків)
  3. Метод Полака–Ріб’єра
  4. Метод Ньютона
  5. Метод Нелдера–Міда (симплексний метод)
  6. Метод Хука–Дживса

- Вибір **будь-якої з вбудованих тестових функцій** (2D):
  1. `f1(x1, x2) = (12 + x1^2 + (1 + x2^2)/x1^2 + ((x1*x2)^2 + 100)/(x1*x2)^4) / 10`
  2. `f2(x1, x2) = (x1 - x2)^2 + (x1 + x2 - 10)^2 / 9`
  3. `f3(x1, x2) = 5 * (x2 - 4*x1^3 + 3*x1)^2 + (x1 + 1)^2`
  4. `f4(x1, x2) = 5 * (x2 - 4*x1^3 + 3*x1)^2 + (x1 - 1)^2`
  5. `f5(x1, x2) = 100 * (x2 - x1^3 + x1)^2 + (x1 - 1)^2`
  6. `f6(x1, x2) = (0.01*(x1 - 3))^2 - (x2 - x1) + exp(20*(x2 - x1))`
  7. `f7(x1, x2) = 100 * (x2 - x1^2)^2 + (1 - x1)^2` (функція Розенброка)
  8. `f8(x1, x2) = (x1 - 4)^2 + (x2 - 4)^2`

- **Табличний вивід** процесу оптимізації:
  - номер ітерації / точки;
  - поточна точка `x`;
  - крок переходу;
  - значення функції `f(x)`.

- **Графічний вивід**:
  - графік залежності `f(k)` від номера ітерації;
  - контурний графік цільової функції з траєкторією руху методу;
  - відображення початкової та фінальної точки.

- **Зведена таблиця результатів** для обраної функції:
  - метод;
  - знайдений мінімум `x*`;
  - значення `f(x*)`;
  - кількість ітерацій;
  - кількість обчислень цільової функції.
